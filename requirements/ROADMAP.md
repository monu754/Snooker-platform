# Project Roadmap

The vision for SnookerStream is to become the premier open-source platform for snooker tournament management and streaming.

## Phase 1: Foundation (Completed)
- [x] Basic authentication (NextAuth)
- [x] Real-time scoring system (Pusher)
- [x] Umpire and Admin dashboards
- [x] Basic match scheduling
- [x] Profile management (Name/Password)

## Phase 2: Engagement (Ongoing)
- [ ] **Live Chat Enhancements**: Adding emojis, user mentions, and moderation tools.
- [ ] **Player Search**: Advanced filtering by name, rank, or country.
- [ ] **Notification System**: Browser alerts for favorite players going live.
- [ ] **Mobile App PWA**: Full offline support for viewing schedules.

## Phase 3: Advanced Features (Planned)
- [ ] **Tournament Brackets**: Automated bracket generation for large events.
- [ ] **Analytics Suite**: Detailed statistics for players (highest break, pot success, etc.).
- [ ] **Multi-Stream Support**: Viewing multiple tables simultaneously on one screen.
- [ ] **API for External Third Parties**: Allow partners to embed live scores.

## Phase 4: Monetization & Pro (Future)
- [ ] **Subscription Tiers**: Premium streams and ad-free viewing.
- [ ] **VOD Library**: Archived matches with chapter markers for big breaks.
